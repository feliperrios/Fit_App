# FitApp
App de fitness (treinos, dieta, dados corporais) com permissoes de coach.

## Stack
- Next.js 14 / React 18 / TypeScript / Tailwind CSS
- Camada de dados abstrata: mock local hoje, pronta para Supabase

## Rodando localmente
```bash
npm install
npm run dev
```

## Deploy no Vercel
1. Importe este repo no Vercel
2. Framework: Next.js (detecta automatico)
3. Configure as env vars do .env.example
4. Deploy
